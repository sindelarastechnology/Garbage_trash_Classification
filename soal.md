# Petunjuk

a. Mahasiswa memilih satu dataset citra dari Kaggle atau dataset publik lainnya.

b. Mahasiswa mencari contoh implementasi program Python yang menggunakan dataset tersebut.

c. Program dapat dijalankan menggunakan:

1. Google Colaboratory  
2. Thonny Python IDE  
3. Jupyter Notebook  

d. Program boleh menggunakan Machine Learning / Artificial Intelligence, tetapi penilaian difokuskan pada konsep Pengolahan Citra Digital yang diterapkan.

e. Mahasiswa membuat video penjelasan berdurasi 7–10 menit dan mengunggah ke Youtube masing-masing.

f. Pengumpulan berupa:

1. Link Youtube.
2. Source code program (.py / .ipynb).
3. Link dataset yang digunakan.
